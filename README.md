## 1. НАЗНАЧЕНИЕ И УСЛОВИЯ ПРИМЕНЕНИЯ ПРОГРАММЫ

### 1.1. Назначение программы

Программа предназначена для расчёта стоимости натяжного потолка для компаний, занимающихся установкой натяжных потолков. Программа позволяет менеджеру или клиенту быстро получить ориентировочную стоимость заказа на основе:

- линейных размеров помещения;
- типа выбранного покрытия;
- цвета покрытия;
- срочности выполнения заказа.

### 1.2. Функции программы

Программа выполняет следующие функции:

1. Загрузка справочной информации о покрытиях и цветах из базы данных.
2. Динамическая фильтрация списка цветов в зависимости от выбранного покрытия.
3. Автоматический расчёт площади потолка по введённым размерам.
4. Расчёт итоговой стоимости заказа с учётом:
   - базовой цены за квадратный метр;
   - наценки 15% за импортное (не российское) покрытие;
   - наценки 20% за срочное выполнение заказа.
5. Информирование пользователя об изменении стоимости при выборе соответствующих параметров.
6. Контроль корректности данных, введённых пользователем.

### 1.3. Условия применения

**Минимальные технические требования к рабочей станции:**

| Параметр | Значение |
|---|---|
| Процессор | Intel Core i3 или аналог (от 2 ГГц) |
| Оперативная память | не менее 2 ГБ |
| Свободное место на диске | не менее 500 МБ |
| Операционная система | Windows 7/8/10/11, Linux (Ubuntu 18.04+) |
| Программное обеспечение | Платформа «1С:Предприятие 8.3» (версия не ниже 8.3.18) |
| Разрешение монитора | не ниже 1024×768 |
| Сетевое подключение | для клиент-серверного варианта работы |

**Требования к программному окружению:**
- установленная платформа 1С:Предприятие 8.3;
- наличие развёрнутой информационной базы со справочниками «Покрытия» и «Цвета»;
- права доступа на чтение указанных справочников.

### 1.4. Требования к квалификации программиста

Программист, осуществляющий сопровождение и модификацию программы, должен обладать следующими знаниями и навыками:

## 2. ХАРАКТЕРИСТИКА ПРОГРАММЫ

### 2.1. Режим работы

Программа работает в интерактивном режиме в среде «1С:Предприятие». Все операции выполняются по запросу пользователя через элементы графического интерфейса формы. Программа не выполняет фоновых заданий и не модифицирует данные информационной базы — работа происходит только в режиме чтения справочной информации.

### 2.2. Временные характеристики

| Операция | Среднее время выполнения |
|---|---|
| Открытие формы калькулятора | до 1 секунды |
| Загрузка списка цветов из справочника | до 0,5 секунды |
| Расчёт стоимости | менее 0,1 секунды |
| Отображение сообщения пользователю | мгновенно |

### 2.3. Средства контроля правильности выполнения

В программе реализованы следующие механизмы контроля:

1. **Предварительная валидация входных данных** — функция `ПроверитьВводДанных()` проверяет заполненность всех обязательных полей и положительность числовых значений перед началом расчёта.
2. **Контроль типов данных** обеспечивается средствами платформы 1С: ввод числовых значений в текстовые поля невозможен.
3. **Обработка исключений** — конструкция `Попытка…Исключение` оборачивает основной блок расчёта и перехватывает непредвиденные ошибки.
4. **Логические проверки** — все ссылочные значения проверяются на заполненность через `ЗначениеЗаполнено()`.
5. **Зависимость справочников** — поле «Цвет» автоматически очищается при смене покрытия, что исключает выбор несовместимых значений.

### 2.4. Средства самовосстановления

- Платформа «1С:Предприятие» обеспечивает автоматическое восстановление сеанса работы при кратковременных сбоях соединения.
- Поскольку программа не модифицирует данные базы (выполняет только чтение справочников), при любом аварийном завершении работы целостность информационной базы не нарушается.
- При возникновении неперехваченной ошибки расчёт прерывается, пользователю выдаётся сообщение, форма остаётся открытой и работоспособной.

### 2.5. Структура программы

Программа реализована в виде объекта метаданных типа «Обработка» с одной формой.

**Используемые объекты конфигурации:**

| Объект | Тип | Назначение |
|---|---|---|
| КалькуляторПотолков | Обработка | Основной модуль программы |
| Покрытия | Справочник | Хранение типов покрытий с ценой и страной |
| Цвета | Справочник | Хранение цветов с привязкой к покрытию |

**Структура реквизитов справочника «Покрытия»:**
- Наименование (Строка, 50)
- СтранаПроизводства (Строка, 50)
- ЦенаЗаКвМ (Число, 10.2)

**Структура реквизитов справочника «Цвета»:**
- Наименование (Строка, 50)
- Покрытие (СправочникСсылка.Покрытия)

## 3. ОБРАЩЕНИЕ К ПРОГРАММЕ

### 3.1. Запуск программы

**Способ 1. Открытие внешней обработки:**
1. Запустить «1С:Предприятие» в режиме «Предприятие».
2. Открыть информационную базу с настроенными справочниками.
3. В главном меню выбрать: **Файл → Открыть**.
4. Указать путь к файлу `КалькуляторПотолков.epf`.
5. Нажать кнопку «Открыть».

**Способ 2. Открытие встроенной обработки:**
1. Запустить «1С:Предприятие» в режиме «Предприятие».
2. В меню «Операции» выбрать пункт «Обработки».
3. В списке выбрать «Калькулятор потолков» и нажать «ОК».

### 3.2. Передача параметров

Программа не принимает внешних параметров при запуске. Все исходные данные вводятся пользователем интерактивно через элементы формы.

### 3.3. Состав процедур и функций модуля формы

| Имя процедуры/функции | Тип | Назначение |
|---|---|---|
| `ПокрытиеПриИзменении` | Процедура | Обработчик события смены покрытия, обновляет список цветов и инициирует проверку импорта |
| `НастроитьОтборЦветов` | Процедура | Заполняет список выбора цветов в зависимости от выбранного покрытия |
| `ЗаполнитьСписокЦветов` | Процедура | Серверная функция формирования запроса к БД для получения цветов |
| `ПроверитьИмпортноеПокрытие` | Процедура | Проверяет страну производства и уведомляет о наценке 15% |
| `ПолучитьСтрануПокрытия` | Функция | Возвращает страну производства покрытия из справочника |
| `СрочностьПриИзменении` | Процедура | Обработчик флажка срочности, информирует о наценке 20% |
| `Рассчитать` | Процедура | Основная процедура расчёта итоговой стоимости |
| `ПроверитьВводДанных` | Функция | Валидация заполненности и корректности полей формы |
| `ПолучитьЦенуПокрытия` | Функция | Возвращает базовую цену за м² выбранного покрытия |

## 4. ВХОДНЫЕ И ВЫХОДНЫЕ ДАННЫЕ

### 4.1. Входные данные

Входные данные вводятся пользователем через форму калькулятора.

| Параметр | Тип | Источник | Ограничения |
|---|---|---|---|
| ШиринаСм | Число (10,0) | Поле ввода | Положительное число > 0 |
| ВысотаСм | Число (10,0) | Поле ввода | Положительное число > 0 |
| Покрытие | СправочникСсылка.Покрытия | Выбор из справочника | Обязательное для заполнения |
| Цвет | СправочникСсылка.Цвета | Список выбора | Обязательное; фильтр по покрытию |
| Срочность | Булево | Флажок формы | Истина/Ложь |

### 4.2. Промежуточные данные (из БД)

| Поле | Тип | Источник | Назначение |
|---|---|---|---|
| Покрытия.ЦенаЗаКвМ | Число (10,2) | Справочник «Покрытия» | Базовая цена за 1 м² |
| Покрытия.СтранаПроизводства | Строка (50) | Справочник «Покрытия» | Признак импортного материала |

### 4.3. Выходные данные

| Параметр | Тип | Назначение |
|---|---|---|
| ИтоговаяСтоимость | Число (15,2) | Результат расчёта в рублях, округлён до копеек |

Дополнительно пользователю выводятся информационные сообщения о применённых наценках в соответствующих окнах сообщений.

### 4.4. Формула расчёта

```
Шаг 1. Перевод размеров в метры:
    Ширина_м = ШиринаСм / 100
    Высота_м = ВысотаСм / 100

Шаг 2. Расчёт площади:
    Площадь = Ширина_м × Высота_м

Шаг 3. Базовая стоимость:
    Стоимость = Площадь × ЦенаЗаКвМ

Шаг 4. Применение наценки за импорт (если СтранаПроизводства ≠ "Россия"):
    Стоимость = Стоимость × 1.15

Шаг 5. Применение наценки за срочность (если Срочность = Истина):
    Стоимость = Стоимость × 1.20

Шаг 6. Округление до копеек:
    ИтоговаяСтоимость = Окр(Стоимость, 2)
```

| «Невозможно выполнить расчёт. Исправьте ошибки:» | Предупреждение | Сводное сообщение о всех ошибках валидации | Исправить указанные пункты и повторить расчёт |
